## Features

前端工程化演示项目:

- 基于 React 框架
- Jest + plugins 实现单元测试
- Eslint + Prettier + Husky 语法检查
- 采用 Tailwindcss 原子化CSS实现
- 利用Monorepo方式管理组件库生态
- 采用 Rollup 构建
- Dumi + Vercel 文档网站搭建
- 基于 Action CI 实现持续集成与交付
